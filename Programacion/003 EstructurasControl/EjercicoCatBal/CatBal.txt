Las estructuras de control nos permiten ejecutar diferentes linias de codigo
de forma condicional, es decir, la ejecucion de esas linias esta sujeta a la
comparacion de unos valores con otros.

Esto sirve para elegir que partes del codigo se aplican segun la situacion

2.
Para introducir una condicion usamos la funcion if
```
if condicion:
   codigo a ajecutar
```
El codigo solo se ejecutara en caso de que la condicions ea verdadera,
por el contrario si es falsa pasara otra cosa que podemos definir con la
funcion else.
```
if condicion:
   codigo a ejecutar
   (caso verdadero)
else:
   codigo a ejecutar
   (caso falso)
```
Para casos en los que tenemos mas de una resolucion para la condicon existe la 
funcion elif (else if). Esta funcion permite poner mas posibles casos que se
pueden llegar a dar y el codigo en respuesta.
El else tambien se puede usar, solo ejecutandose cuando no se cumpla ninguna de
las otras condiciones.
```
if condicon1:
   codigo a ejecutar1
elif codicion2
   codigo a ejecutar2
else:
   codigo a ejecutar
```
