
En esta actividad se practica la asignación de valores a variables de forma correcta y como mostrarlaspor pantalla junto a más texto. Como aficionado a los videojuegos se que esto e suna parte muy importantepara definir el estado de varias cosas.

Las variables son asignaciones de valores de distintos tipos a identificadores. 
Para declarar variables necesitamos un identificador, un operador de asignación y un valor.
```
repeticiones = 3
```
Luego para mostrar las variables por pantalla se usa print.
Lo que se encuentra entre comillas se imprimirá de forma normal y lo que está entre comas es la variable.
```
print("Hoy hize",repeticiones,"repeticiones de sentadilla")
```

A la hora de poner nombre a las variables hay que asegurarse de evitar algunos errores.
```
1edad_jugador = 25                    #No puede comenzar con un número
nivel-dificultad = "medio"           #El guion (-) no está permitido
puntos acumulados = 1000         #Los espacios no están permitidos
def = 5                                         #"def" es una palabra reservada de Python
```
