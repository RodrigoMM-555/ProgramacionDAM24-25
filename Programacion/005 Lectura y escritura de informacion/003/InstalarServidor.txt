
Instalar un servidor web en entorno de desarrollo (el propio ordenador)

- Windows:
Se recomienda instalar el paquete XAMPP
htpps//:www.apachefriends.org/es/index.html

Una vez instalado, la carpeta de publicacion es c:/xampp/htdocs

- Linux
Se instala Apache con el comando
sudo apt install apache2

una vez instalado, la carpeta de publicacion es : /var/www/html

Como se suele ahcer en linux se recomienda que se ponga en preactica en la maquina virtual
