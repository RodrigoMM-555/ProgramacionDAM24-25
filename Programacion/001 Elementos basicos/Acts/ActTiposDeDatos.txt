Los tipos de datos son fundamentales a la hora de definir las variables. Escoger un tipo de valor incorrecto puede llevar a errores e inconsistencias en el código.
En esta actividad nos aseguraremos de que los datos sean del tipo correcto.

2.
De los tipos de datos en Python, en este ejercicio usamos los enteros (int) aunque también conocemos los flotantes (float)
```
peras=int(input("Cuantas peras tienes:"))
```
De esta manera nos aseguramos de que la edad introducida sea un número entero. En caso de ser un umero decimal, int lo redondearía y en caso de ser una cadena de texto el programa daría error.
```
altura=float(input("Dame tu altura:"))
```
En este caso, float mantendrá los decimales en caso de haberlos. De igual manera, dará error si el dato es una cadena de texto.
Una vez el dato es del tipo que queremos, podemos operar con ellos. En este caso con int podemos efectuar operaciones matemáticas.
```
fruta=peras+manzanas
```
Con que solo un valor sea float el resultado, aunque sea entero, se considerara float
